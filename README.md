# Hotel-Reservations-Analysis

## Overview
This project analyses hotel reservations, and build an association between guest booking patterns and their high cancellation rates. Phase 1 includes Data Pre-processing and EDA, while Phase 2 solely focuses on build machine learning models.

## Repository Structure
```
.
├── Phase 1 Cleaning and Analysis.ipynb
├── Phase 2 Modelling.ipynb
└── README.md
```

## Data Availability
The dataset can be found in Kaggle, see references.

## Requirements
- Python 3.x

## Getting Started
1. Clone the repository.
2. Download the datasets, see references.
3. Run Phase 1 and then Phase 2 code
